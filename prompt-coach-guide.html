Hashtag Attributes Explained

Positive Attributes (Green)

These indicate your prompt follows best practices:

#persona #context #format #examples #constraints #specific #structured #templated
  • #persona - Defines a role (e.g., "You are a senior developer...")
  • #context - Provides background information or scenario
  • #format - Specifies desired output format
  • #examples - Includes examples for clarity
  • #constraints - Sets boundaries (must, should, don't)
  • #specific - Contains specific details or numbers
  • #structured - Uses markdown formatting (lists, headers)
  • #templated - Has placeholders like [variable] or {{variable}}

Warning Attributes (Yellow/Red)

These suggest areas for improvement:

#too-brief #verbose #needs-structure #vague
  • #too-brief - Prompt is under 10 words; add more detail
  • #verbose - Prompt exceeds 500 words; consider simplifying
  • #needs-structure - Missing persona, context, or format instructions
  • #vague - Could benefit from specific details or examples

The CO-STAR Framework

A proven structure for writing effective prompts:

  • C - Context: Background information the AI needs
  • O - Objective: The specific task you want completed
  • S - Style: Writing style or tone to use
  • T - Tone: Emotional quality (formal, casual, enthusiastic)
  • A - Audience: Who the output is intended for
  • R - Response: Desired format (list, essay, code, JSON)

Quick Tips

  • Be specific: "Write 5 bullet points" beats "write some points"
  • Use delimiters: Triple quotes, XML tags, or markdown to separate sections
  • Specify format: Tell the AI exactly how you want the output structured
  • Provide examples: Show the AI what good output looks like
  • Iterate: Refine your prompt based on results
